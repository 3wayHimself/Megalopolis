# Megalopolis

This repository contains the portable deep learning library implementation for .NET platform. This library is written by C#.

## Features

* .NET Standard 2.0 library
* Code first modeling
* Dependency-free

### Activation Functions
* Hyperbolic tangent
* Identity
* ReLU
* Sigmoid
* Softmax

### Layers
* Convolutional
* Dropout
* Fully connected
* Pooling

### Loss Functions
* Cross-entropy
* Mean squared error (MSE)

### Optimizers
* AdaDelta
* AdaGrad
* Momentum
* RMSprop
